# Ganu Vibes – Terengganu Tourism Event Portal

## Overview
**Ganu Vibes** is an interactive tourism website designed to showcase the best of Terengganu, Malaysia. The platform allows users to explore cultural attractions, natural landscapes, and unique tourism packages while providing an easy-to-use online booking experience. This project aims to promote Terengganu as a top travel destination for both local and international tourists.


### 1. Homepage
- Welcomes visitors with a visually appealing banner of Terengganu.
- Provides navigation links to **Packages**, **Events**, **Login**, and **Registration**.
- Offers an overview of the website and highlights key tourism packages.

### 2. Packages Section
- Displays a list of tourism packages with images, descriptions, destinations, duration, and pricing.
- Users can click **Book Now** to proceed with bookings.

### 3. Booking Page
- Users can select a package, choose a date, and enter the number of tickets.
- Displays current bookings with an option to cancel via a **Delete** button.

### 4. Registration and Login
- Allows new users to create an account or existing users to log in for bookings.

### 5. Search Feature
- Enables users to search for specific event packages based on their preferences.

### 6. Profile Update
- Users can update their profile information such as name.
- Ensures user profiles remain up-to-date and secure.

### 7. Review Section
- Users can leave reviews and feedback for the packages they have booked.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Django (Python)
- **Database:** SQLite
